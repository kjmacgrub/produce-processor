# Produce Processing App

Web-based produce processing tracker for Park Slope Food Co-op.

## Deployment

This repository is automatically deployed to Netlify. Any push to the `main` branch triggers a new deployment.

**Live URL:** https://[your-site-name].netlify.app

## Files

- `index.html` - Main processing app (v1.11)
- `pdf-uploader.html` - Admin PDF uploader (v1.11)

## How to Update

1. Download updated files from Claude
2. Upload to this repository (drag & drop on GitHub website)
3. Commit changes
4. Netlify automatically deploys within ~30 seconds
