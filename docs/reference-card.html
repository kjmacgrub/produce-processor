Produce Processor

Emergency reference · current as of 2026-04-22

What it is

An iPad-based web app used in the produce department to work through a daily processing task list. A receiving manager uploads a CSV each morning; iPads on the floor pull from a shared list in real time, letting staff mark items complete, record short instructional videos, and capture completion photos.

If it's broken — start here

  1. Site loads but data is stale → Firebase console → Realtime Database, confirm today's items are present under /items.
  2. CSV didn't arrive → Firebase Storage bucket process-6d2dc.firebasestorage.app, path produce-csv/YYYY-MM-DD.csv.
  3. Site is down → Netlify dashboard → Deploys tab. Latest deploy should be green; rollback if red.
  4. Videos won't play → check Storage path produce-videos/{item_id}.webm.
